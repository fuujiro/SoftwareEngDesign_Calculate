## web科学计算器

环境要求：python 3.x
* 如何查看本地python环境：
    1. 进入cmd
    2. 输入 python --version
    3. 如出现如 "Python 3.6.5 :: Anaconda, Inc." ，其他python3以上版本皆可

使用步骤:
1. 进入cmd
2. 路径进入SoftwareEngDesign_Calculate/code下（win是\）
3. 输入 python calculate.py 和 python app.py
1. 如果没有报错，按照命令行提示，在浏览器中网址栏输入http://127.0.0.1:5000/
2. 即可使用计算器
